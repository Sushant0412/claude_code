# GitHub Integration
This agent has access to the GitHub MCP server. 

## Instructions
- When asked to "push," "pull," or "create a PR," use the `github-custom` MCP tools.
- Use `search_repositories` to find relevant code in my other repos.
- Always verify the branch name before pushing.

## Shortcuts
- /push : Commits all changes and pushes to the current branch.
- /pr : Creates a new pull request with a summary of the current changes.
